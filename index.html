🚀 TRICKLE AI CORE

Умный контроль интернета и sing-box для роутеров

⚡ Что это

TRICKLE AI CORE — автономная система для роутеров, которая следит за интернетом, автоматически перезапускает VPN и отправляет уведомления в Telegram.

🚀 Установка

cd /opt && wget https://raw.githubusercontent.com/vidar87/trickle-ai-core/main/install.sh && sh install.sh

🎛 Управление

Запуск панели управления:

tricklectl

📊 Возможности

Auto Restart Latency Monitor Telegram Alerts Web UI Low CPU

🌐 Веб-панель

После установки открой:

http://IP-ROUTER:8787

🔄 Обновление

sh /opt/trickle-ai/update.sh

🧹 Удаление

sh /opt/trickle-ai/uninstall.sh

🧠 Для чего

Для нестабильного интернета, VPN соединений и роутеров, которые должны работать 24/7 без вмешательства.
